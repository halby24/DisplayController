# DisplayController 使用ガイド

## 概要
DisplayControllerは、環境光センサーを使用してモニターの明るさを自動調整するデーモンプログラムです。SwitchBotライトセンサーやその他のプラグイン型センサーに対応しています。

## インストール方法
1. リリースページから最新のバイナリをダウンロード
2. 必要なプラグインを`plugins`ディレクトリに配置
3. `config.json`を設定（設定方法は[configuration.md](configuration.md)を参照）

## 起動方法
```bash
./DisplayController
```

## 動作確認
1. プログラムが正常に起動することを確認
2. ログに環境光センサーの値が表示されることを確認
3. モニターの明るさが環境光に応じて自動調整されることを確認

## トラブルシューティング
- プログラムが起動しない場合：
  - 設定ファイルの構文を確認
  - プラグインが正しく配置されているか確認
  - ログファイルを確認

- センサー値が取得できない場合：
  - センサーの接続状態を確認
  - プラグインの設定を確認
